## Description

ตรวจหวย — เครื่องมือตรวจผลสลากกินแบ่งรัฐบาลที่ครบครันที่สุด!

ดูผลรางวัลล่าสุดได้ทันทีจาก Chrome toolbar ไม่ต้องเปิดเว็บ พร้อมข้อมูลย้อนหลังกว่า 5 ปี (120+ งวด) และเครื่องมือวิเคราะห์สถิติครบทุกรูปแบบ

# Privacy & Permissions

## Single Purpose Description
ตรวจผลสลากกินแบ่งรัฐบาลและวิเคราะห์สถิติตัวเลข

## Permission Justifications

### storage
ใช้เพื่อ cache ข้อมูลผลรางวัลย้อนหลังไว้ใน localStorage เพื่อให้เปิดใช้งานได้เร็วขึ้นโดยไม่ต้องโหลดข้อมูลจาก API ทุกครั้ง

### alarms
ใช้เพื่อเช็คเวลาทุก 1 นาทีสำหรับแสดง badge บน icon ของ extension เมื่ออยู่ในช่วงวันออกรางวัล (วันที่ 1 และ 16 ของเดือน)

### Host Permission: https://lotto.api.rayriffy.com/*
ใช้เพื่อดึงข้อมูลผลรางวัลสลากกินแบ่งรัฐบาลจาก API สาธารณะ ซึ่งเป็นแหล่งข้อมูลผลรางวัลอย่างเป็นทางการ

## Data Usage Disclosure
- ไม่เก็บข้อมูลส่วนตัวของผู้ใช้
- ไม่ส่งข้อมูลไปยังเซิร์ฟเวอร์ภายนอก (นอกจาก API ผลรางวัล)
- ข้อมูลทั้งหมดเก็บอยู่ใน browser ของผู้ใช้เท่านั้น (localStorage)
- ไม่มีการติดตามพฤติกรรมผู้ใช้ (no analytics, no tracking)

# Privacy Policy

## Privacy Policy for Thai Lottery Chrome Extension

**Last updated: February 16, 2026**

### Information We Collect
This extension does NOT collect, store, or transmit any personal information.

### Data Storage
- Lottery results data is cached locally in your browser's localStorage for performance
- No data is sent to external servers other than fetching lottery results from the public API (lotto.api.rayriffy.com)
- All data remains on your device

### Permissions Used
- **storage**: To cache lottery results locally for faster loading
- **alarms**: To check draw times and update the extension badge icon
- **Host permission (lotto.api.rayriffy.com)**: To fetch official lottery results

### Third-Party Services
- Lottery data is fetched from lotto.api.rayriffy.com, a public API for Thai lottery results
- No analytics, advertising, or tracking services are used

### Changes to This Policy
We may update this privacy policy from time to time. Changes will be reflected in the "Last updated" date.
